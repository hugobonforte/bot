# El prefijo para que el bot te haga caso es el signo de exclamación invertido: ¡

Aquí tienes los comandos que puedes escribir en el chat de Discord:

¡hola - El bot te saludará y te dirá su nombre.

¡heh - El bot escribirá "hehe..." varias veces. Si pones un número después (por ejemplo ¡heh 10), lo repetirá esa cantidad de veces.

¡kodland - El bot enviará un mensaje de bienvenida a la escuela de Kodland.

¡suma - Debes poner dos números después del comando para que el bot los sume. Ejemplo: ¡suma 5 8 (el bot responderá 13).

¡escoge - Escribe varias opciones separadas por espacios y el bot elegirá una por ti. Ejemplo: ¡escoge perro gato pájaro.

¡memepy - El bot buscará una imagen en tu carpeta de archivos llamada "images" y la enviará al chat.

¡meme - El bot buscará una imagen en tu otra carpeta llamada "image" y la enviará al chat.

Recuerda que para que los últimos dos funcionen, debes tener las carpetas con esos nombres y fotos adentro en tu computadora.
